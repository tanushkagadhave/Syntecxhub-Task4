🏏 IPL Data Analysis using Python

A data analysis project on Indian Premier League (IPL) match data to explore player performance, team success, and season-wise trends using Python.

📌 Features

Top run scorers analysis

Strike rate calculation

Team win percentage analysis

Season-wise player performance

Detailed performance analysis of Rohit Sharma

Data visualizations using Matplotlib

Exported summary insights

📂 Dataset

This project uses two CSV datasets:

matches.csv – match-level information

deliveries.csv – ball-by-ball delivery data

Public IPL datasets commonly used for data analysis projects.

🛠️ Tech Stack

Python

Pandas

Matplotlib

Jupyter Notebook / VS Code

🔍 Analysis Breakdown
1. Top Scorers

Identifies the top 10 players based on total runs scored.

2. Strike Rate Analysis

Calculates strike rate for each player to identify high-impact batters.

3. Team Win Rates

Computes win percentages for all teams across seasons.

4. Season-wise Performance

Analyzes player performance trends across different IPL seasons.

5. Rohit Sharma Performance

Total runs and strike rate

Season-wise run analysis

Trend visualization

📊 Visualizations

Bar charts for:

Top run scorers

Strike rates

Team win rates

Line charts for:

Player performance across seasons

📁 Output

visual_summary.csv – consolidated performance insights

Interactive graphs displayed during execution

▶️ How to Run
pip install pandas matplotlib

Run the Python script or Jupyter Notebook
